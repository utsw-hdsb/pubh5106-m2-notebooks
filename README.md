# Module 2: Pre-Class Exploration Notebooks

**PUBH 5106 — AI in Health Applications**

These interactive notebooks accompany the Module 2 pre-class video lectures. They are designed to build hands-on intuition about how LLMs process text.

## Notebooks

| Notebook | Topic | Paired With | Time |
|----------|-------|-------------|------|
| `M2_explore_tokenization.ipynb` | BPE tokenization, subword splitting, clinical text | Segment B | ~15 min |
| `M2_explore_embeddings.ipynb` | Word embeddings, GloVe, distributional semantics | Segment B | ~25 min |
| `UDL_12_3_Tokenization.ipynb` | Tokenization deep-dive (BPE algorithm step-by-step) | Segment B | ~20 min |
